# 4 Servo Motors Sweep

## Tinkercad Simulation
https://www.tinkercad.com/things/aqmygnnrZzn/editel?sharecode=sYUnfCYgjVNx5G4fk8ZRLdH37uk99M8D0Ce836Qd3Jo

## Description
This project controls four servo motors using Arduino Uno.

The four servo motors perform a sweep motion for 2 seconds and then stop at 90 degrees.

## Components
- Arduino Uno
- 4 Micro Servo Motors
- Jumper Wires

## Connections
- Servo 1 → Pin 3
- Servo 2 → Pin 5
- Servo 3 → Pin 6
- Servo 4 → Pin 9

All servo VCC pins are connected to 5V.

All servo GND pins are connected to GND.

## Files Included
- 4_Servo_Motors_Sweep.ino (Arduino Code)
- Circuit Screenshot

## Result
All four servo motors perform a sweep motion for 2 seconds and then hold at 90 degrees.
